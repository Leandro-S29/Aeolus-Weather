# Aeolus Weather
> Minimal desktop app built with CustomTkinter to deliver real-time weather insights powered by OpenWeatherMap.

---

## ✨ Features
- Live weather data for any city worldwide.
- Temperature, humidity, wind speed, and condition summaries.
- User friendly interface built with CustomTkinter.

---

## 🚀 Getting Started
### Prerequisites
- Python 3.11+
- OpenWeatherMap API key

### Installation
1. Clone the repository  
   ```sh
   git clone https://github.com/Leandro-S29/Aeolus-Weather.git
   ```
2. Install dependencies  
   ```sh
   pip install customtkinter python-dotenv requests Pillow
   ```
3. Create a `.env` file inside `Aeolus Project/`:
   ```sh
   OpenWeatherKey="YOUR_API_KEY"
   ```
